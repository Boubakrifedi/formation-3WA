Au cours de cette formation nous allons avoir 2 matiéres :

L'intégration

Le HTML:

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS:

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, etc.

Le développement

Le Javascript:

Javascript est un language de programmation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramétres.

Le PHP:

Le PHP est un language de programmtion libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certains informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc.)

MYSQL:

MySQL est un systéme de gestion de base de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de création de compte) pour ensuite être ré-utilisé au moment approprié (ex: connexion à un site internet avec identifiant et mot de passe).

400 heures de formation intensive :

Le programme de formation de la 3W Academy est découpé en 57 modules.

  • HTML, CSS, Responsive web desgin, Wordpress, Bootstrap : 16 modules
  • Javascript : 12 modules
  • PHP, MySQL, POO, MCV: 25 modules
